پایتون چیست؟
پایتون یک زبان برنامه‌نویسی سطح بالا، تفسیری و همه‌منظوره است که به خاطر خوانایی بالا و سادگی نحوی مشهور است. از پایتون در توسعه وب، علم داده، هوش مصنوعی و خودکارسازی استفاده می‌شود.

چگونه یک متغیر در پایتون تعریف کنیم؟
در پایتون نیازی به اعلام نوع متغیر نیست. کافیست اسم متغیر را نوشته و مقداردهی اولیه را با علامت مساوی (=) انجام دهید. مثال: name = "Ali"

حلقه for در پایتون چگونه کار می‌کند؟
حلقه for برای پیمایش روی یک توالی (مثل لیست، رشته یا range) استفاده می‌شود. نحو آن: for item in sequence: و سپس ادامه کد با تورفتگی.

چگونه یک تابع در پایتون تعریف کنیم؟
با کلمه کلیدی def و سپس نام تابع و پرانتز (ممکن است پارامتر داشته باشد) و دو نقطه، بدنه تابع با تورفتگی نوشته می‌شود. مثال: def sum(a, b): return a + b

چطور یک لیست در پایتون ایجاد کنیم؟
لیست‌ها با کروشه [] تعریف می‌شوند. مثال: fruits = ["apple", "banana", "cherry"]

کتابخانه Flask چیست؟
Flask یک میکرو فریم‌ورک تحت وب برای پایتون است که برای ساخت برنامه‌های وب ساده و سبک استفاده می‌شود. با Flask می‌توان به سادگی مسیرها (routes) و صفحات را تعریف کرد.

چگونه یک برنامه ساده با Flask بسازیم؟
ابتدا کتابخانه flask را نصب کنید. سپس فایلی مثل app.py ایجاد کنید و در آن یک اپلیکیشن تعریف کرده و یک مسیر برای نمایش سلام دنیا بنویسید. سپس با اجرای flask run سرور را راه‌اندازی کنید.

RAG چیست؟
RAG (Retrieval-Augmented Generation) روشی است که در آن یک مدل زبانی بزرگ هنگام تولید پاسخ، اطلاعات مرتبط را از یک پایگاه داده متنی (مانند اسناد) بازیابی می‌کند و سپس پاسخ را بر اساس آن اطلاعات تولید می‌کند. این کار باعث می‌شود پاسخ‌ها دقیق‌تر و مستندتر شوند.

چگونه می‌توان داده‌ها را در پایتون از یک فایل خواند؟
با استفاده از تابع open() می‌توان فایل را باز کرده و محتویات آن را با متد read() یا readlines() خواند. بهتر است از بلوک with استفاده کنید تا فایل به‌طور خودکار بسته شود.

Git چیست و چرا استفاده می‌شود؟
Git یک سیستم کنترل نسخه است که به برنامه‌نویسان اجازه می‌دهد تغییرات کد را ردیابی کنند و به صورت تیمی روی پروژه‌ها کار کنند. با Git می‌توان به راحتی به نسخه‌های قبلی برگشت.
